2. 일상생활의 데이터베이스
? 데이터베이스
: 조직에 필요한 정보를 얻기 위해 논리적으로 연관된 데이터를 모아 구조적으로 통합해 놓은 것

- 데이터베이스 시스템은 데이터의 검색과 변경작업을 주로 수행함
- 변경이란 시간에 따라 변하는 데이터값을 데이터베이스에 반영하기 위해 수행하는
  삽입, 삭제, 수정 등의 작업

  * 통합된 데이터 (Integrated Data) : 데이터를 통합하는 개념, 각자 사용하던 데이터의 중복을 최소화하여 중복으로 인한 데이터 불일치 현상을 제거
  * 저장된 데이터 (Stored Data) : 문서로 보관된 데이터가 아니라 디스크, 테이프 같은 컴퓨터 저장장치에 저장된 데이터 
  * 운영 데이터(Operational Data) : 프로그램의 목적을 위해 사용되는 데이터. 업무를 위한 검색을 할 목적으로 저장된 데이터  
  * 공용 데이터(Shared Data) : 한 사람 또는 한 업무를 위해 사용되는 데이터가 아니라 공용으로 사용되는 데이터

  3. 데이터베이스의 개념
  - 데이터베이스는 운영 데이터를 통합하여 저장하며 공용으로 사용
